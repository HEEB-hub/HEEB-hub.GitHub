Low-DimensionTool Documentation

This is the Low-Dimension Tool (LDT), which is a self-developed dimension reduction utility based on the open-source CFD platform OpenFOAM.

LDT performs a dimension reduction procedure based on the proposed Self-adaptive Low-dimension Ventilation Model.

The proposed Self-adaptive Low-dimension Model, including direct and automatic modes, together with an advanced non-uniform dividing module, is able to execute a series of low-dimension conversions of given high-resolution scalar field.

At the end of the processing, a new low-dimension mesh and corresponding scalar field will be constructed in the ./LD folder , which can be easily accessed by OpenFOAM (i.e. ParaView) for visualization.

Methods implemented

  • lowDimensionMethod
    • import (CFD scalar field & mesh)
    • execution of low-dimension process
    • low-dimension error calculation
    • advanced non-uniform dividing method
    • low-dimension magnitude update
    • export (Low-Dimension mesh & low-dimension scalar field)
